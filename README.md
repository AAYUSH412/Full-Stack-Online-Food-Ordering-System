## 🚀 Quick Start Guide

### Prerequisites
- Node.js
- MongoDB
- npm/yarn

### Installation Steps
1. Clone Repository
   ```bash
   git clone https://github.com/yourusername/bitebuddy.git
   ```

2. Setup Backend
   ```bash
   cd backend
   npm install
   ```

3. Setup Frontend
   ```bash
   cd frontend
   npm install
   ```

4. Configure Environment
   - Create `.env` files
   - Add necessary credentials

5. Launch Development Servers
   ```bash
   # Backend
   npm run server

   # Frontend
   npm run start
   ```
